Student365

wat is het?

Een app die buitenlandse studenten kunnen gebruiken om een studieplek te vinden in Amsterdam op basis van de voorkeuren die zij hebben.

Bekijk de website Bekijk de code

focus

Web Development, UI/UX

tools

HTML, CSS, Javascript, Jquery

het doel

Het doel van de opdracht was om een specifiek probleem op te lossen die studenten uit het buitenland ervaren wanneer zijn komen studeren in Nederland. Dit kon in de vorm van een applicatie, een spel, een app en voor welk probleem je zelf wilde.

Voor mijn eigen applicatie heb ik er voor gekozen om te focussen op het vinden van een goede studieplek die aansluit bij de behoeften van de student in Amsterdam.

de uitwerking

Voor de uitwerking heb ik gekozen voor een mobiele applicatie waar de student opzoek kan gaan naar zijn/haar studieplek. De gebruiker kiest wat voor voorzieningen hij/zij wil voor de studieplek. Op basis van deze keuzes krijg je een paar resultaten met studieplekken waar je doorheen kan 'browsen'.
